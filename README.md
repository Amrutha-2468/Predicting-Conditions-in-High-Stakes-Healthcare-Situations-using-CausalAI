# Predicting-Conditions-in-High-Stakes-Healthcare-Situations-using-CausalAI
AI-powered healthcare system using Causal AI for disease prediction, X-ray analysis, and real-time health monitoring.

## 📌 Overview

This project focuses on improving decision-making in critical healthcare scenarios using **Causal Artificial Intelligence (Causal AI)**. Unlike traditional machine learning models that rely on correlations, this system identifies **cause-and-effect relationships** to provide more accurate and reliable predictions.

The application integrates **symptom-based disease prediction, X-ray image analysis, and real-time health monitoring**, making it suitable for high-risk environments like ICUs and emergency care.

---

## 🚀 Features

* 🔍 Symptom-based disease prediction using Machine Learning
* 🩻 X-ray image analysis for respiratory diseases
* 📊 Real-time health data visualization (charts & dashboards)
* 🧠 Causal AI for better decision-making
* 🔐 User authentication and secure data handling
* 📝 Medical history tracking
* 📩 Alert system for critical conditions

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/project-name.git
cd project-name
```

---

### 2. Install dependencies

#### For frontend & backend

```bash
npm install
```

#### For ML model (Python)

```bash
pip install -r requirements.txt
```

---

### 3. Run the project

#### Start frontend (Next.js)

```bash
npm run dev
```

#### Start backend / API

```bash
npm run start
```

#### Run ML Flask server

```bash
python app.py
```
